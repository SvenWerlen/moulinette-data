# Moulinette Tokens Packs

This folder contains the required data for generating token packs.
* [Samples](samples/README.md) : Samples for testing [Moulinette Tokenizer](https://github.com/SvenWerlen/moulinette/tree/master/tokenizer/README.md)
* [Pathfinder](pathfinder/README.md) : data packs for Pathfinder Adventure Paths
* [People](people/README.md) : People, folks, thugs, villagers, etc.
