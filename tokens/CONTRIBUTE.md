# Moulinette Tokenizer

This page describes **How to contribute** by submiting new token packs.

## Prerequisites

* You must install [Moulinette TokenTool](https://github.com/SvenWerlen/moulinette-tokentool). Supports Linux and Windows only for now.

## Steps to create a bundle of images (novice)

If you want to contribute:
* Download the [Moulinette TokenTool](https://github.com/SvenWerlen/moulinette-tokentool)
* Prepare a data pack by choosing your images from Internet (see [Moulinette TokenTool](https://github.com/SvenWerlen/moulinette-tokentool) for instructions)
* Export the pack and share the JSON file by either
  * Opening an [issue](issue)
  * Sharing it on [Discord](https://discord.gg/xg3dcMQfP2)


## Steps to create a bundle of images (advanced contributor)

* Same instructinos as above

Then:

* Create a pull request with the manifest file
* Submit and wait for approval
